##Project Overview##
This project is a Django-based Social Media Web Application.
It allows users to interact through posts, likes, comments, stories, and chat.
The system also includes an admin dashboard to monitor platform activity.

##Key Features##
User authentication (Login, Signup, Logout)
Role-based access control (Admin and User)
Post management (Create, Read, Update, Delete)
Like and Comment system
Follow/Unfollow users
Stories feature
Chat system between users
Profile with cover image
Admin dashboard with real-time data (users, posts, likes, comments, messages)
